POST http://localhost:8080/api/v1/users
Content-Type: application/json

{
  "phoneNumber": "+919876543211",
  "displayName": "Jane Doe",
  "about": "Available",
  "profilePictureUrl": "https://cdn.example.com/jane.jpg"
}

Step 3 — John adds Jane as a Contact
POST http://localhost:8080/api/v1/contacts
Content-Type: application/json

{
  "ownerId": 1,
  "contactUserId": 2,
  "nickname": "Jane Work"
}
Expected: 201 Created → isBlocked: false

Step 4 — Create a Private Chat between John and Jane
POST http://localhost:8080/api/v1/chats/private?senderId=1&receiverId=2
No body needed. Expected: 201 Created → note down id: 1 (chatId)

Step 5 — John sends a Text Message
POST http://localhost:8080/api/v1/messages
Content-Type: application/json

{
  "chatId": 1,
  "senderId": 1,
  "content": "Hey Jane! How are you?",
  "messageType": "TEXT"
}
Expected: 201 Created → status: SENT → note down id: 1 (messageId)

Step 6 — Jane replies to John's Message
POST http://localhost:8080/api/v1/messages
Content-Type: application/json

{
  "chatId": 1,
  "senderId": 2,
  "content": "I am great! Thanks for asking.",
  "messageType": "TEXT",
  "replyToMessageId": 1
}
Expected: 201 Created → replyToContent: "Hey Jane! How are you?"

Step 7 — Mark John's Message as Delivered
PATCH http://localhost:8080/api/v1/messages/1/status?status=DELIVERED
No body needed. Expected: 200 OK → deliveredAt timestamp set

Step 8 — Mark John's Message as Read
PATCH http://localhost:8080/api/v1/messages/1/status?status=READ
No body needed. Expected: 200 OK → readAt timestamp set

Step 9 — Get all Messages in Chat (verify everything)
GET http://localhost:8080/api/v1/messages/chat/1
Expected: 200 OK → 2 messages in order, message 2 has replyToMessageId: 1

Step 10 — Create a Group
POST http://localhost:8080/api/v1/groups
Content-Type: application/json

{
  "name": "Dev Team",
  "description": "Our dev group",
  "createdById": 1,
  "memberIds": [2]
}
Expected: 201 Created → John = ADMIN, Jane = MEMBER → note down id: 1 (groupId)

Step 11 — Create Group Chat
POST http://localhost:8080/api/v1/chats/group/1
No body needed. Expected: 201 Created → chatType: GROUP → note down id: 2 (group chatId)

Step 12 — John sends a Message in Group Chat
POST http://localhost:8080/api/v1/messages
Content-Type: application/json

{
  "chatId": 2,
  "senderId": 1,
  "content": "Welcome to Dev Team group!",
  "messageType": "TEXT"
}

Step 13 — John posts a Status
POST http://localhost:8080/api/v1/statuses
Content-Type: application/json

{
  "userId": 1,

Step 14 — Jane views John's Status

Content-Type: application/json

[1]

GET http://localhost:8080/api/v1/chats/user/2
