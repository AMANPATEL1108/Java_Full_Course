🚀 STEP 1 — Register 3 Users
Register Alice:
POST http://localhost:8080/api/auth/register
Content-Type: application/json

{
  "name": "Alice Sharma",
  "email": "alice@example.com",
  "password": "alice123",
  "phone": "9876543210"
}
Register Bob:
POST http://localhost:8080/api/auth/register
Content-Type: application/json

{
  "name": "Bob Patel",
  "email": "bob@example.com",
  "password": "bob123",
  "phone": "9123456780"
}
Register Charlie:
POST http://localhost:8080/api/auth/register
Content-Type: application/json

{
  "name": "Charlie Singh",
  "email": "charlie@example.com",
  "password": "charlie123",
  "phone": "9012345678"
}

🔑 STEP 2 — Login (do all 3, save tokens)
Login Alice:
POST http://localhost:8080/api/auth/login
Content-Type: application/json

{
  "email": "alice@example.com",
  "password": "alice123"
}
Login Bob:
POST http://localhost:8080/api/auth/login
Content-Type: application/json

{
  "email": "bob@example.com",
  "password": "bob123"
}
Login Charlie:
POST http://localhost:8080/api/auth/login
Content-Type: application/json

{
  "email": "charlie@example.com",
  "password": "charlie123"
}

From each login response, save: data.token and data.userUuid
You'll get 3 tokens and 3 UUIDs. Call them ALICE_TOKEN, BOB_UUID, CHARLIE_UUID etc.


👤 STEP 3 — Verify Profile (paste Alice's token)
GET http://localhost:8080/api/users/me
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

👥 STEP 4 — Create Group (as Alice)
POST http://localhost:8080/api/groups
Content-Type: application/json
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

{
  "name": "Goa Trip 2025",
  "description": "Trip expenses for Goa vacation"
}

From response save: data.uuid → call it GROUP_UUID


➕ STEP 5 — Add Members (as Alice, paste GROUP_UUID)
Add Bob:
POST http://localhost:8080/api/groups/PASTE_GROUP_UUID_HERE/members
Content-Type: application/json
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

{
  "userUuid": "PASTE_BOB_UUID_HERE"
}
Add Charlie:
POST http://localhost:8080/api/groups/PASTE_GROUP_UUID_HERE/members
Content-Type: application/json
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

{

}

GET http://localhost:8080/api/groups/my
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

🧾 STEP 7 — Add Expenses
Expense 1 — EQUAL split, Alice paid hotel ₹3000:
POST http://localhost:8080/api/expenses
Content-Type: application/json
Authorization: Bearer PASTE_ALICE_TOKEN_HERE

{
  "description": "Hotel booking",
  "amount": 3000.00,
  "category": "TRAVEL",
  "splitType": "EQUAL",
  "groupUuid": "PASTE_GROUP_UUID_HERE",
  "paidByUserUuid": "PASTE_ALICE_UUID_HERE"

Tokens → from each login response data.token
UUIDs of users → from each register/login response data.userUuid
Group UUID → from create group response data.uuid
